
| **Statistic**          | **Formula**                                     |
| ---------------------- | ----------------------------------------------- |
| **Mean**               | \( \mu = \frac{1}{n} \sum x_i \)                |
| **Median**             | Middle value                                    |
| **Mode**               | Most frequent value                             |
| **Range**              | \( x_{\text{max}} - x_{\text{min}} \)           |
| **Variance**           | \( \sigma^2 = \frac{1}{n} \sum (x_i - \mu)^2 \) |
| **Standard Deviation** | \( \sigma = \sqrt{\sigma^2} \)                  |
| **IQR**                | \( Q_3 - Q_1 \)                                 |
