### GenAI 是一種通用技術
GenAI：LLM + UI X
ex: 電剛剛出現的時候，大家只會做電燈泡，卻沒有想到其他的可能
GenAI 是通用技術
每個角落每個人都可以用上
就跟用電一樣，GenAI 百工百頁都可以用到

GenAI vs. 其他通用技術

| GenAI    | 其他       |
| -------- | -------- |
| 只要會講話就可以 | 需要學習     |
| 進步速度很快   | 進步速度很慢   |
| 只需要雲端    | 需要很多硬體設備 |
